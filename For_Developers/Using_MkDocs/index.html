Install mkdocs if you haven’t already, here are the instructions. For full documentation visit mkdocs.org.

View Docs locally

Run mkdocs serve from the root project directory, this starts up a local server to which you can connect using your browser, usually at http://localhost:8000/. The server watches for changes and automatically refreshes the site.

There is no need to use the mkdocs build command, this creates all the html files needed for deployment and we use a different method for that.
